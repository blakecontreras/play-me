# play-me
An app for digital discovery

## Roadmap

### Front-end
* Framework: Vue.js
* State manager: Vuex

#### Views
* User Dashboard
  * Favorite Game
  * Links to Queues
  * Top Suggestion
  * Search Games
* Rating Queue
* Recommendation Queue
* Demo Version
* Splash Page / Welcome Page

### Server
* Giant Bomb API
* Steam API (maybe)
* Battle.net API (maybe)
* User Authentication
* Store Ratings in DB

### Database
* Postgres for user info
* Neo4j for recommendations
